magento-pt_br by mirago

Este repositório contém o Magento e seus principais aplicativos utilizados no Brasil. Aqui você encontrará todos os arquivos do Magento já com os pacotes de tradução para português (Brasil) e os principais módulos de pagamento, entrega e outros.Para saber mais, clique na aba "Wiki" acima.

Install

Você pode instalar este pacote de duas maneiras: 1) Faça o download dos arquivos e depois faça o upload para sua hospedagem 2) Através do Git faça um clone em seu servidor local e depois suba para sua hospedagem Consulte a página Wiki sobre instalação, clicando no menu superior na aba "Wiki".
